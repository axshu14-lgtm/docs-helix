Below is every command available to regular members of the Helixia server. All commands start with the `.` prefix.

## Profile & Information

| Command        | Aliases                  | What It Does                                                                                   |
| -------------- | ------------------------ | ---------------------------------------------------------------------------------------------- |
| `.info`        | `.i`, `.balance`, `.bal` | View your profile — balance, boxes, messages, vanity time, tag time, and level.                |
| `.help`        | —                        | Browse all available commands in a clean grid.                                                 |
| `.serverstats` | `.srvstats`, `.stats`    | View server-wide statistics like total messages, coins in circulation, and level distribution. |

## Economy

| Command           | Aliases                    | What It Does                                                                                    |
| ----------------- | -------------------------- | ----------------------------------------------------------------------------------------------- |
| `.openbox`        | `.ob`                      | Open one or more mystery boxes for random Pokécoin rewards. Use `.openbox 5` or `.openbox all`. |
| `.helpbox`        | `.hb`                      | Learn how mystery boxes work and how to earn them.                                              |
| `.sendbal`        | `.sb`, `.transfer`, `.pay` | Send Pokécoins to another user. Minimum 50,000 PC.                                              |
| `.deposithistory` | `.dh`                      | View your deposit and claim history.                                                            |

## Daily Quests

| Command   | Aliases               | What It Does                                              |
| --------- | --------------------- | --------------------------------------------------------- |
| `.quests` | `.dq`, `.dailyquests` | View your daily quest progress, streak info, and rewards. |

## Gambling

| Command                | Aliases                          | What It Does                                           |
| ---------------------- | -------------------------------- | ------------------------------------------------------ |
| `.gamble @user amount` | `.g`, `.bet`, `.duel`            | Challenge another user to a Pokécoin gambling duel.    |
| `.gamblehelp`          | `.ghelp`, `.gh`                  | Learn the rules for every gambling game type.          |
| `.gamblingprofile`     | `.gp`, `.gprofile`               | View your personal gambling statistics.                |
| `.gamblingleaderboard` | `.glb`, `.gleaderboard`, `.gtop` | View gambling leaderboards across multiple categories. |
| `.gamblestats`         | `.gstats`                        | View server-wide gambling statistics.                  |

## Events

| Command | Aliases | What It Does                                          |
| ------- | ------- | ----------------------------------------------------- |
| `.roll` | —       | Roll a random number (1–100). Used during Roll Event. |

## Progression

| Command        | Aliases       | What It Does                                                         |
| -------------- | ------------- | -------------------------------------------------------------------- |
| `.leaderboard` | `.lb`, `.top` | View leaderboards for messages, vanity, balance, gambling, and more. |
| `.leveluplist` | `.lvllist`    | View all configured level-up rewards.                                |

## Community

| Command         | Aliases | What It Does                                                                 |
| --------------- | ------- | ---------------------------------------------------------------------------- |
| `.vanitystatus` | `.vs`   | Check your vanity status, time tracked, and whether it's currently counting. |
| `.tagstatus`    | `.ts`   | Check your server tag status and time tracked.                               |
| `.sh`           | —       | View shiny hunters.                                                          |

<Info>
  Achievements are viewed through the `.info` command — click the **Achievements** button after running `.info` to see your progress.
