## What is Helix?

Helix is a feature-rich Discord community built for the Helixia Pokémon community. It powers the server's economy, daily quests, events, gambling system, achievement tracking, and much more — all centered around earning and spending **Pokécoins (PC)**.

Whether you're completing daily quests, gambling against friends, opening mystery boxes, or climbing the leaderboards, this documentation covers everything you need to know.

## Quick Reference

All commands use the `.` prefix. For example, `.info` will show your profile.

| What you want to do   | Command                 |
| --------------------- | ----------------------- |
| View your profile     | `.info`                 |
| Check daily quests    | `.quests`               |
| Open mystery boxes    | `.openbox`              |
| Start a gamble        | `.gamble @user amount`  |
| Check vanity status   | `.vanitystatus`         |
| View leaderboards     | `.leaderboard`          |
| Send coins to someone | `.sendbal @user amount` |
| Get help              | `.help`                 |

<Tip>
  You can use shorthand amounts like `50k` for 50,000 or `1m` for 1,000,000 when entering Pokécoin values.
</Tip>

## Where to Use Commands

Most commands can only be used in the **designated** [commands](https://discord.com/channels/1459854752580898868/1459854755378499793) **channel**. If you try to use a command in the wrong channel, the bot will let you know where to go.

There are two exception(s):

- **Gambling commands** (`.gamble`, etc.) have their own dedicated gambling channels.
- **`.roll`** can be used in the Roll Event channel during active events.
